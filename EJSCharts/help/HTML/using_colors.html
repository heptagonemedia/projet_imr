Using Colors

 

Colors may be specified in any of the following formats:

 

RGB:        "rgb( <red>, <green>, <blue> )"

 

       example: "rgb(255,0,0)"

 

RGBA:        "rgba( <red>, <green>, <blue>, <opacity> )"

 

       example: "rgba(255,0,0,50)"

 

HEX:        "#<red><green><blue>"

 

       example: "#ff0000"
