EJSC.Chart.x_min

See Also

 

Definition

 

float x_min = undefined

 

Description

 

Defines the current minimum X value of the chart.

 

Example

 

>> Force the x-axis range to extend beyond the data it contains

 

var chart =  new EJSC.Chart(

              "chart",

              {x_min: -100.00}

       );
