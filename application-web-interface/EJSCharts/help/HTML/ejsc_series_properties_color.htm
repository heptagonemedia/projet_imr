EJSC.Series.color

See Also

 Top  Previous  Next

 

Definition

 

string color = undefined

 

Description

 

Defines the series color.  If left undefined, the color is pulled from the array of default colors stored in the chart.

 

       Note: May not be applicable to all series (i.e. EJSC.PieSeries)
