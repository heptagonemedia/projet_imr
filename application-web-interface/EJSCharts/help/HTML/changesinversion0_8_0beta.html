Changes in version 0.8.0 Beta

Additions

 

•Added chart.show_titlebar property and chart.showTitlebar()/chart.hideTitlebar() methods
•Added support for [label] replacement in pie series custom hints
•Added chart.show_x_axis property and chart.showXAxis()/chart.hideXAxis() methods
•Added chart.show_y_axis property and chart.showYAxis()/chart.hideYAxis() methods
•Added "power" trend series type

 

Modifications

 

•Pie series default hints now include series title
•Modified show_options to __show_options, functionality is not fully implemented
•Renamed decimal_seperator to decimal_separator and thousand_seperator to thousand_separator in EJSC.NumberFormatter

 

Bug Fixes

 

•Fixed trend and function series legend icons
•Fixed x_zero_plane to correctly read properties
•Fixed chart redraw method to resize drawing area if necessary
•Fixed pie series reading EJSC.DefaultPieColors
•Fixed bar chart drawing when all y values are > 0
•Fixed text label recognition in series data when label begins with a number
•Fixed bar series legend image to correctly display vertical bar icon
•Fixed chart title to force left align by default instead of inheriting from parent element
•Fixed drawing when axis are not displayed (i.e. pie series by itself).  Chart now encompasses the entire available area.
•Fixed pie series hints to display piece labels if defined
•Fixed legend item visibility when series is initially set to hidden
•Fixed issue with IE resizing chart area incorrectly
