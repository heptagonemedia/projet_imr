EJSC.Chart.getZoom

See Also

 

Definition

 

object getZoom( )

 

       RETURNS:

               { x_min: float, x_max: float, y_min: float, y_max: float }

 

Description

 

Returns the current zoom coordinates in chart units.

 

To set the zoom coordinates, see EJSC.Chart.setZoom
