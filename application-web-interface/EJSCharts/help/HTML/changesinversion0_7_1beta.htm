Changes in version 0.7.1 Beta

Additions

 

•Added trend series with support for linear trends
•Added CSVFileDataHandler class
•Added CSVStringDataHandler class
•Added setLineWidth method to LineSeries, BarSeries, FunctionSeries, AreaSeries
•Added setPointStyle method to ScatterSeries
•Added setDefaultColors method to PieSeries
•Added property to show/hide grid lines (show_grid)
•Added __draw_series_on_load property (default false) to prevent series from redrawing the chart when initially loaded
•Added onBeforeZoom event handler
•Added onBeforeMove event handler
•Added support for onShowMessage to accept boolean return value to cancel message from being shown
•Added public redraw() method to Chart
•Added way to mix forced_xtremes and x_axis_extremes_ticks
•Added setColor method to Series
•Added showGrid, hideGrid methods to Chart
•Added onBeforeZoom event to Chart
•Added onBeforeMove event to Chart
•Added chart as parameter to onBeforeDblClick
•Added series and chart as parameters to onDblClickPoint
•Added support for compact xml format
•Added onPieceNeedsColor event to PieSeries

 

Modifications

 

•Moved __utility namespace to be public
•Updated legend visibility icons to disappear if allow_interactivity = false
•Modified onBeforeVisibilityChange to send chart and series objects
•Modified onAfterVisibilityChange to send chart and series objects and accept boolean return value to cancel redraw
•Modified Series __color to be public property (color) and default value to undefined
•Modified Series __visible to be public property (visible)
•Modified Chart onBeforeDraw method to send chart object
•Updated auto series color generation to check undefined instead of ''                                
•Modified onBeforeUnselectPoint event to send series object
•Modified onAfterUnselectPoint event to send point, series, and chart objects
•Updated showCrosshairs to setCrosshairs and added visible parameter
•Modified onShowCrosshairs to accept boolean return value to cancel crosshairs from being shown
•Updated mouse position to not update if allow_interactivity = false
•Moved initial determination of xml type
•Turned on invalid xml file notification
•Reworked xml parsing logic to account for new compact xml type and remove unnecessary switch checks
•Modified LineSeries, BarSeries, FunctionSeries, AreaSeries __lineWidth to public property (lineWidth)
•Modified draw code in AreaSeries to support faster series drawing with thousands of points
•Modified ScatterSeries __pointSize and __pointStyle to be public properties (pointSize, pointStyle)
•Modified PieSeries __defaultColors to public property (defaultColors)

 

Bug Fixes

 

•Added check for canvas width/height 0
•Reversed y_zero_plane and x_zero_plane functionality
•Updated Bar series to make sure plot width is at least 1 pixel
•Removed drag/select cancelling since it affected the entire document
