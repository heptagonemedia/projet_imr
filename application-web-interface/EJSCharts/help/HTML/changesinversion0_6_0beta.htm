Changes in version 0.6.0 Beta

 Top  Previous  Next

Additions

 

•Added setDataHandler to series in order to allow easy modification of the data handler after series creation
•Added hiding hint (if not sticky) to zooming
•Added base onAfterDataAvailable to series classes
•Added variables for cursor position caption text
•Added variables for hint value caption text (X Value:, Y Value:)
•Added onShowHint event with the ability to specify the contents of the hint window
•Added special text check/replacement for hint data ([chart_title],[series_title],[xaxis],[yaxis],[x],[y])
•Added option to force tick marks at chart edges (expanding scale as required)

 

Changes

 

•Updated date formatter to forget about timezone and use UTC instead
•Changed NumberFormatter to default to no decimal place limitation
•Updated onBeforeSelectPoint onAfterSelectPoint to send point, series, chart, hint element, and select/hover
•Hide cursor position labels and data fields if cursor position captions are undefined or blank
•Added resizing data area to write and resize functions

 

Bug Fixes

 

•Updated color arrays to be copied instead of references, this fixes issues with running out of colors
•Fixed issue with removing/reloading series so that the extremes are recalculated to account for the new data taking a smaller or difference range
•Fixed issue in bar and line/scatter/area series where the first point was lost if x == 0
•Fixed __doDrawAxis to disregard timezone
•Added check to NumberFormatter to check if first digit is thousand_seperator
•Fixed extreme value recalc on series data reload
