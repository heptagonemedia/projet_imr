Changes in version 0.5.3 Beta

 Top  Previous  Next

Additions

 

•Added the ability to cancel point selection via onBeforeSelectPoint event handler
•Added onBeforeDblClick handler
•Added handler to mouseout for crosshairs
•Added function to show crosshairs instead of mousemove doing it
•Added handler to mousemove for crosshairs
•Added saving X and Y cursor position for use in handler for crosshairs
•Added call to crosshairs handler
•Added allow_zoom variable
•Added proximity_snap to allow for more/less snapping to find points
•Added reload function to all applicable series
•Added setYExtremes/setXExtremes to force extreme values
•Added a way to programatically find the closest point to a given coordinate
•Added support for static labels to the ArrayDataHandler

 

Changes

 

•Updated all __selectNextPoint and __selectPreviousPoint series functions to call chart’s __selectPoint function
•Updated select point to ensure previous/next series is visible
•Updated __findPointInRange to select true closest point
•Updated line, area, and scatter series to allow hints to show text values
•Updated bar series to allow hints to show text values

 

Bug Fixes

 

•Resize doesn’t fire unless the actual width or height changes
•Changed load of excanvas.js to ensure load before drawing in IE.
•Fixed acquireSeries to delete only the requested series from old chart’s array
•Updated mousemove function to call showCrosshairs function
