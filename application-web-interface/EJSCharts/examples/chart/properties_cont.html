Properties Demonstrated

  • x_axis_size
  • x_axis_className

Description

In the original chart the X axis labels overlapped because they were too long to all fit on one line. The chart below makes use of the x_axis_size property to set the axis height to 40 pixels (default axis label height is 20 pixels, 40 pixels = lines desired * label height). In addition, the x_axis_className property is used to apply a CSS style to the X axis caption.

  • x_axis_size

Description

Display a static number of ticks (3) on the X axis instead of using the auto tick positioning. Modify the tick labels to be a different font, bold and gray, and remove the X axis caption to allow for more room to draw the chart and tick labels.

Properties Demonstrated

  • x_axis_minor_ticks
  • y_axis_minor_ticks

Description

Display minor tick marks for the x and y axis. X axis minor ticks should be very transparent and extend the full height of the chart, y axis minor ticks should be less transparent and only extend 3 pixels from the edge.
