Desired Result

This chart will have not be interactive (ie zooming, moving, point selection, etc.) and auto axis scaling for the x axis will be disabled. In addition the status messages (such as "Drawing..." when the chart is loading) will not be displayed and the title will display custom text.

Properties Demonstrated

  • x_axis_extremes_ticks
  • x_axis_formatter
  • x_cursor_position_caption

Desired Result

This chart will display a custom x axis caption, display tick marks at the x axis minimum and maximum points, format the x axis data as currency, and will specify custom text to be displayed before the cursor position. In addition it will not show hints when a point is hovered over or selected.
